## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. The site is fully responsive and works on desktop, tablet, and mobile

## Customization

### Colors
Edit the CSS variables in `css/styles.css`:
```css
:root {
    --primary-color: #1a1a1a;
    --secondary-color: #d4a574;
    --accent-color: #e8d4c0;
    --text-color: #333;
    --light-bg: #f5f5f5;
}
```

### Content
Update the HTML content in `index.html` with:
- Band name and tagline ✓
- About section description
- Album/EP information
- Gallery images
- Contact details
- Social media links

### Images
Add your band photos to the `images/` folder and update the image paths in `index.html`
